# OccuBias Dataset

OccuBias is a structured prompt-based dataset designed to evaluate occupational gender bias in large language models.

## Dataset Overview
- Total prompts: 6,300
- Professions: 35
- Occupational categories: 7
- Template types: Neutral, Ambiguous, Stereotype breaking

## Columns
- category
- profession
- template_type
- bio


## Associated Research
This dataset accompanies the MSc by Research thesis:
**"Occupational Gender Bias in Large Language Models: A Multi-Level Analysis Using the OccuBias Dataset"**
