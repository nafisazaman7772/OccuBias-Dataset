# OccuBias Dataset

OccuBias is a structured prompt-based dataset designed to evaluate occupational gender bias in large language models.

## Dataset Overview
- Total prompts: 6,300
- Professions: 35
- Occupational categories: 7 (e.g., STEM, Healthcare, Legal, etc.)
- Template types: Neutral, Ambiguous, Stereotype-breaking

## Columns
- category: Occupational group (e.g., STEM, Healthcare)
- profession: Specific job role (e.g., Doctor, Engineer)
- template_type: Type of prompt (Neutral, Ambiguous, Stereotype-breaking)
- bio: Biographical prompt used as model input

## Template Types
- Neutral: Describes a profession without explicit gender cues  
- Ambiguous: Includes contextual workplace interactions without clear gender  
- Stereotype-breaking: Challenges traditional gender-role associations  

## Usage
This dataset can be used to:
- Evaluate gender bias in LLM outputs  
- Analyse bias across professions and contexts  
- Support research in AI fairness and responsible AI  

## Associated Research
